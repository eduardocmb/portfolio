El Camino Recorrido

Cada etapa ha sido una pieza fundamental en mi evolución como desarrollador.

2021

Primer contacto con el código

Mis primeras líneas en HTML y CSS. Descubrí que podía construir cosas reales con código y no lo solté desde entonces.

2021 — 2022

Desarrollo Web Frontend

Profundicé en JavaScript moderno, diseño responsive y la creación de interfaces dinámicas e interactivas.

2022

Backend con Laravel

Aprendí a construir APIs robustas, sistemas de autenticación y aplicaciones completas con el framework más elegante de PHP.

2022 — 2023

Desarrollo Móvil con Flutter

Descubrí el poder de Flutter para crear apps multiplataforma con una sola base de código. Dart se convirtió en mi segundo lenguaje favorito.

2023

Proyectos Reales y C# / .NET

Comencé a trabajar en proyectos para clientes reales mientras expandía mis capacidades hacia el ecosistema Microsoft con C# y .NET.

2024 — Presente

Full Stack en Producción

Desarrollo soluciones completas end-to-end para empresas, combinando backend sólido, frontend moderno y apps móviles fluidas.
